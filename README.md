# Medico - Clinic Management System

A comprehensive clinic management system built with Laravel for managing patients, doctors, appointments, medical records, and test results.

## STEPS AFTER CLONING AND TESTING

**1. Install PHP Dependencies:**

composer install


**2. Install Node.js Dependencies:**

npm install


**3. Configure Environment (.env):**

copy .env.example or cp .env.example .env


**4. Run Database Setup:**

php artisan migrate

php artisan db:seed


**5. Build assets and serve:**

npm run dev

php artisan serve

**6. Visit the page:**

visit the page on (localhost)/login
